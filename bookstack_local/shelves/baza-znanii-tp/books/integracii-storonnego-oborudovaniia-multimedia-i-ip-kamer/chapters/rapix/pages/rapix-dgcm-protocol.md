# RAPIX DGCM протокол

RAPIX использует протокол DGCM (Digital General Control Module) для управления освещением.

## Транспорт

- TCP порт 36689
- Формат: JSON строка + `\r\n`
- Также поддерживает REST HTTP и WebSocket

## Аутентификация

Не требуется для базовых команд на TCP. Для HTTP/WS может потребоваться токен.

## Формат команд

```json
{"type":"dgcm","ver":1,"id":<request_id>,"cat":"<category>","cmd":"<command>","data":[...]}
```

Ответ:
```json
{"type":"dgcm","ver":1,"id":<request_id>,"cat":"<category>","cmd":"<command>","status":<code>,"data":[...]}
```

## Команды зон (cat: "zone")

| Команда | data | Описание |
|---------|------|----------|
| `zone_list` | `[]` | Получить список всех зон |
| `zone_info` | `["<zone_id>"]` | Информация о зоне |
| `on` | `["<zone_id>"]` | Включить зону |
| `off` | `["<zone_id>"]` | Выключить зону |
| `fade_to_level` | `["<zone_id>", <level>, <fade_time_ms>]` | Установить уровень (0-255 или 0.0-100.0) |
| `fade_to_color` | `["<zone_id>", ..., <fade_time_ms>]` | Установить цвет |

## Команды сцен (cat: "xiene")

| Команда | data | Описание |
|---------|------|----------|
| `set` | `["<scene_id>"]` | Активировать сцену |
| `nudge` | `["<scene_id>", <zone_id>, <level>]` | Подкорректировать зону в сцене |
| `offset` | `["<scene_id>", <zone_id>, <offset>]` | Смещение зоны в сцене |

## Команды системы (cat: "system")

| Команда | data | Описание |
|---------|------|----------|
| `identify` | `[]` | Идентификация контроллера (мигание LED) |
| `ping` | `[]` | Проверка связи |
| `extended_status` | `[]` | Статус с диагностикой |

## Подписка на события (cat: "event")

```json
{"type":"dgcm","ver":1,"id":3,"cat":"event","cmd":"subscribe","data":["/zone/+/level","/zone/+/on","/xiene/+/set"]}
```

## Ответ с событиями

```json
{"type":"dgcm","ver":1,"id":0,"cat":"event","cmd":"","status":200,"data":[{"event":"/zone/1/level","data":[50]}]}
```

## Полезные статус-коды

- 200 — успех
- 400 — неверный запрос
- 404 — зона/сцена не найдена
- 500 — внутренняя ошибка контроллера

## Источник

Тикет 591-266500, документация RAPIX, эксперименты с прототипом драйвера.
