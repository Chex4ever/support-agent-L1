# Настройка автозапуска iRidi Server через Планировщик задач Windows

## Обзор
Данная инструкция описывает процесс настройки автоматического запуска iRidi Server в ОС Windows. Использование Планировщика задач позволяет серверу запускаться в фоновом режиме сразу после загрузки системы, обеспечивая бесперебойную работу без необходимости входа пользователя в систему.

## Подготовка
Для корректной работы iRidi Server требуется запуск приложения с правами администратора. Планировщик задач гарантирует, что сервер будет активен независимо от текущей сессии пользователя.

## Пошаговая настройка

### 1. Запуск Планировщика
Нажмите `Win + R`, введите `taskschd.msc` и нажмите **Enter**.

### 2. Создание задачи (Вкладка General)
1. На панели справа выберите **Create Task** (Создать задачу).
2. В поле «Name» введите имя, например: `iRidiServerAutostart`.
3. На вкладке **General** укажите следующие параметры:
   - **Run whether user is logged on or not** (Выполнять для всех пользователей) — вариант для фоновой работы (консоль сервера не будет отображаться, рекомендуется при наличии пароля у УЗ пользователя на ПК или наскольких УЗ).
   - **Run only when user is logged on** (Выполнять для пользоавтелей, вошедших в систему) - только если при включении учетная запись автоматически входит в систему, консоль сервера будет отображаться
   - **Run with highest privileges** (Выполнить с наивысшими правами) — **обязательно** для корректной работы сервера.
   - **Configure for** — выберите текущую версию ОС Windows.

### 3. Настройка триггеров (Вкладка Triggers)
1. Перейдите на вкладку **Triggers** и нажмите **New**.
2. В поле «Begin the task» выберите **At startup** (При запуске системы).
3. В блоке «Advanced settings» можно активировать **Delay task for** (например, на 30 секунд), чтобы дождаться полной инициализации сетевых служб ОС.
4. Убедитесь, что галочка **Enabled** установлена.

### 4. Конфигурация действий (Вкладка Actions)
1. Перейдите на вкладку **Actions** и нажмите **New**.
2. Action: **Start a program**.
3. Program/script: нажмите **Browse** и выберите исполняемый файл `iRidiServer.exe` в папке установки.

### 5. Параметры запуска (Вкладки Conditions и Settings)
- **Вкладка Conditions**: Снимите все флажки, особенно «Start the task only if the computer is on AC power», чтобы сервер не останавливался при работе от аккумулятора.
- **Вкладка Settings**: 
  - Оставьте активными: «Allow task to be run on demand» и «Run task as soon as possible...».
  - В поле «If the task is already running, then...» выберите **Do not start a new instance**.

## Проверка
После сохранения задачи выполните следующие шаги:
1. Нажмите правой кнопкой мыши на созданную задачу в списке и выберите **Run**.
2. Убедитесь, что сервер запустился.
3. Перезагрузите компьютер и проверьте наличие процесса `iRidiServer.exe` в Диспетчере задач после загрузки системы.
