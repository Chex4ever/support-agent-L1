# Таблица соответствия: как пользователь называет → что это на самом деле

## Путаница в названиях

| Как часто говорят пользователи | Что на самом деле имеют в виду | Комментарий |
|-------------------------------|-------------------------------|-------------|
| "iRidium" | iRidi Pro или любое ПО iRidi | Пользователи называют компанию и продукты как угодно |
| "i3 pro" | Клиентское приложение для панелей | Часто говорят "айридиум", "айриди", "апп" |
| "iRidium server" | Может означать и софт (серверный демон), и железо (HS Server) | Уточнять: про программу или про коробку? |
| "Bus77 Home" | Это **ПО** для настройки Bus77 Pro оборудования, а не железо! | Самая частая путаница |
| "Bus77 Lite" | Это **ПО** для настройки Bus77 Lite оборудования | Тоже софт, а не железо |
| "Басовка", "Басс" | Bus77 протокол | Сленг от "Bus" |
| "Lite" | Bus77 Lite (софт + оборудование) или i3 Lite (старое) | Уточнять: про освещение (Lite) или про упрощенную версию? |
| "Pro" | iRidi Pro (экосистема) или Bus77 Pro (оборудование) | Контекст: про визуализацию или про железо? |
| "HS Server" | Конкретная железка | Пользователи редко знают точную модель |
| "Сервер" | iRidium server (софт) или HS Server (железо), или NUC/RPi | Всегда уточнять |
| "UMC" | Старое оборудование (C2, C3, C4) | Сейчас заменяется на HS Server |
| "i3 lite" | Устаревшая версия, архив | Не путать с Bus77 Lite |
| "Панель", "Touch panel" | Или железная панель (P6/P7/P8/P10), или iPad с i3 pro | Уточнять модель |
| "Проект", "панелька" | `.irpz` файл — проект панели | |
| "Серверный проект" | `.sirpz` файл — проект сервера | |
| "Transfer", "Трансфер" | iRidium transfer — утилита для загрузки | |
| "Облако" | iRidium Cloud | |
| "Ключ", "USB-ключ" | Аппаратный оффлайн-ключ лицензии | |
| "HWID" | Hardware ID — привязан к железу | Проблема: может меняться при обновлениях |

## Связанные продукты и их синонимы

```
iRidi (компания) = iRidium Mobile (старое название)
iRidi Pro (экосистема) ≠ i3 pro (только клиент)
Bus77 (протокол) ≠ Bus77 Home (софт) ≠ Bus77 Lite (софт) ≠ Bus77 Pro/Lite (оборудование)
HS Server = Bus77 Home Server = Bus77 Integration Server = iRidi Pro Server = KNX Home Server
  (один и тот же hardware, разная прошивка/лицензия)
```

## Алгоритм уточнения продукта

1. Пользователь говорит "у меня X" → проверить по таблице выше
2. Если ambiguity → задать уточняющие вопросы из `triage_questions.md`
3. Проверить: софт или железо?
4. Проверить: если Bus77 — Pro или Lite?
5. Проверить: если iRidi Pro — studio, i3 pro, server, transfer?
