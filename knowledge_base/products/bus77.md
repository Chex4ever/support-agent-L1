# Bus77 — протокол автоматизации

## Общее описание

Bus77 — российский запатентованный (патент №2653231) CAN-протокол автоматизации.

## Технические характеристики

- **Физический уровень**: CAN (ISO 11898), 125 кбит/с
- **Архитектура**: децентрализованная полевая шина
- **Макс. устройств**: до 127 на сегмент
- **Шифрование**: AES-256 / "Кузнечик" (ГОСТ)
- **Open source**: протокол опубликован на GitHub

## Программное обеспечение для настройки

| Приложение | Для какого оборудования | Макс. устройств |
|-----------|------------------------|-----------------|
| **Bus77 Home** | Bus77 **Pro** оборудование | 64 (рекомендуется), до 127 |
| **Bus77 Lite** | Bus77 **Lite** оборудование | 32 (рекомендуется), до 127 |

## Линейки оборудования

| Характеристика | Bus77 Pro | Bus77 Lite |
|---------------|-----------|------------|
| Рынок | Luxury homes, коммерция | Квартиры комфорт-класса |
| Цвет | Черный | Белый |
| Гарантия | 5 лет | 2 года |
| Макс. устройств | 64 (127) | 32 (127) |
| Сервер | HS Server + iRidi Pro / Bus77 Home Server | Bus77 Lite Server |
| ПО настройки | Bus77 Home или iRidi Pro | Bus77 Lite |

## Важно!
- **Bus77 Home и Bus77 Lite — это ПО, а не оборудование!** Пользователи часто путают.
- Оборудование Pro и Lite **НЕ взаимозаменяемо**.
- Нельзя подключить Lite-устройства через Bus77 Home и наоборот.
