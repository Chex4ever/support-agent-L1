# iRidi Pro

## Общее описание

**iRidi Pro** — флагманская платформа управления для объектов любой сложности: от частных домов до коммерческих зданий. Включает среду разработки, серверное ПО, клиентские приложения и облачные сервисы.

## Состав экосистемы

| Компонент | Назначение | Платформы |
|-----------|-----------|-----------|
| **iRidium studio** | IDE для разработки GUI и конфигурации серверных проектов | Windows |
| **i3 pro** | Клиентское приложение, загружает и исполняет `.irpz` проекты | iOS, Android, Windows, macOS |
| **iRidium transfer** | Инструмент загрузки интерфейсов на панели управления | Windows |
| **iRidium server** | Серверный демон: БД, логика, шлюз между протоколами | Разные платформы |
| **iRidium Cloud** | Облачное хранение лицензий и распространение проектов | Web |

## Форматы проектов

| Формат | Описание |
|--------|----------|
| `.irpz` | Стандартный проект панели |
| `.pirpz` | Зашифрованный проект панели |
| `.sirpz` | Серверный проект |

## Лицензирование

- **Триал**: 5 минут при регистрации, 60 дней через iRidium Cloud
- **Коммерческие**: на панель + на сервер
- **Хранение**: iRidium Cloud для лицензий и обновлений проектов
- **Оффлайн**: аппаратный USB-ключ

## Поддерживаемые протоколы

KNX, Modbus, DALI, Z-Wave, Zigbee, BACnet, HDL, Crestron, AMX, Helvar, Domintell, MQTT, Lutron, Global Cache, Bus77, и др.
