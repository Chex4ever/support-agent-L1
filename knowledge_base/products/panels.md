# Панели управления

## Touch-панели

| Модель | Описание |
|--------|----------|
| P6 | 6-дюймовая сенсорная панель |
| P7 | 7-дюймовая сенсорная панель |
| P8 | 8-дюймовая сенсорная панель |
| P10 | 10-дюймовая сенсорная панель |

Все панели имеют встроенное управление Bus77 CAN.

## Серия AV-R
- Специализированные панели для AV-решений

## Программные панели (i3 pro)
Любое устройство с i3 pro может работать как панель управления:
- iOS (iPad, iPhone)
- Android (планшеты, телефоны)
- Windows (ПК, touch-мониторы)
- macOS
