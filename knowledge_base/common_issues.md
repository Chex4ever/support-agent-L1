# Частые проблемы по категориям

## 1. Лицензирование и активация

| Проблема | Возможные причины | Решение / Ссылка |
|----------|------------------|------------------|
| "Wrong ID" при активации | Неверный HWID, смена оборудования | Проверить HWID, запросить новый лицензионный файл |
| HWID меняется при каждом запуске | Виртуальная машина, обновление ОС | iRidium server не поддерживает VM! Проверить на физическом железе |
| "License file corrupted" | Поврежден файл лицензии | Запросить новый файл в iRidium Cloud |
| Не активируется триал | Уже был активирован, проблема с облаком | Проверить аккаунт в iRidium Cloud |
| Путаница: device license vs site license | Разные типы лицензий | Device license — на конкретное устройство; Site license — на объект |

## 2. Загрузка проекта / Transfer

| Проблема | Возможные причины | Решение / Ссылка |
|----------|------------------|------------------|
| Panel search не находит панель | Другая подсеть, WiFi, firewall | Проверить IP, подсеть, отключить firewall |
| "Authorization is false" | Неверные учетные данные | Проверить логин/пароль интегратора |
| Transfer не видит устройства | Порты заняты (8888), torrent-трекеры | netstat -an найти занятые порты |
| Проект не загружается | Неправильный WiFi, статический IP | Настроить статический IP на панели |
| Firewall/антивирус блокирует Transfer | Блокировка портов | Добавить в исключения |

## 3. Серверные проблемы

| Проблема | Возможные причины | Решение / Ссылка |
|----------|------------------|------------------|
| "ERROR CORE project file does not exist" | Нет серверного проекта (.sirpz) | Загрузить серверный проект |
| Web-интерфейс сервера недоступен | Сервер не запущен, не тот порт | Проверить статус сервера, порт |
| Ошибка авторизации на сервере | Неверные учетные данные интегратора | Проверить credentials |
| Сервер не стартует на VM | iRidium server не поддерживает VM | Установить на физическое железо |
| Медленная работа сервера | Слишком много тэгов | Оптимизировать проект, проверить лимиты |

## 4. Сетевые проблемы

| Проблема | Возможные причины | Решение |
|----------|------------------|---------|
| Панели не находят сервер | Разные подсети, нет маршрутизации | Настроить VLAN, проверить L3 |
| TCP порты не отвечают | Конфликт портов (8888) | netstat -anp \| grep LISTEN |
| Потеря связи с устройствами Bus77 | Обрыв CAN-шины, терминирующие резисторы | Проверить физическое подключение |
| Не работает обнаружение в iRidium Transfer | Несколько сетевых адаптеров | Отключить лишние адаптеры |

## 5. Интеграция по протоколам

| Протокол | Частые проблемы |
|----------|----------------|
| **HDL** | HDL Scanner не находит устройства — проверить подсеть |
| **Modbus TCP** | Неправильный unit ID, неверный порт |
| **KNX** | Неправильные group addresses, нет связи с KNX gateway |
| **BACnet** | Не отображаются точки — проверить BBMD |
| **DALI** | Нет связи с DALI gateway |
| **MQTT** | Неправильный topic, нет broker |

## 6. Аппаратные проблемы

| Проблема | Возможные причины | Решение |
|----------|------------------|---------|
| HS Server не включается | Неправильный блок питания | HP Server требует 30W (не 2.5W как UMC C3!) |
| CAN-шина не работает | Нет терминирующих резисторов | Установить 120 Ом на концах шины |
| Панель P6/P7/P8/P10 не загружается | Неправильная прошивка | Перепрошить через recovery |
| RPi3 зависает | Перегрузка (>15 панелей) | Уменьшить кол-во панелей или перейти на NUC |

## 7. Проблемы с приложением i3 pro

| Проблема | Возможные причины | Решение |
|----------|------------------|---------|
| i3 pro падает после загрузки проекта | Поврежден .irpz файл | Пересобрать проект в iRidium studio |
| Черный экран в i3 pro | Несовместимая версия | Обновить i3 pro или пересобрать проект |
| Нет звука | Не настроен audio driver | Проверить настройки драйвера |

## 8. Legacy / Миграция

| Проблема | Решение |
|----------|---------|
| Старый проект V1.5/V2.0 | Мигрировать на актуальную версию iRidi Pro |
| i2Control (старое приложение) | Больше не поддерживается — перейти на i3 pro |
| UMC C2/C3/C4 | Заменить на HS Server (внимание: разница в питании и разъемах!) |
