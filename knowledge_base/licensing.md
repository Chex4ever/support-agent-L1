# Лицензирование iRidi

## Общая информация

### Типы лицензий

- **Device license** — на конкретное устройство (привязана к HWID)
- **Site license** — на объект (несколько устройств)

### Триал

- 5 минут тестовой лицензии при регистрации
- 60 дней через iRidium Cloud

### Периферийное лицензирование (iRidi Pro)

Каждая панель + сервер требуют отдельную лицензию.

### Серверное лицензирование (iRidi SCADA)

Лицензия на сервер, без привязки к панелям.

## Хранение лицензий

- **iRidium Cloud** — облачное хранение и распространение
- **Оффлайн**: аппаратный USB-ключ

## Частые проблемы

| Проблема | Причина | Решение |
|----------|---------|---------|
| "Wrong ID" | Несовпадение HWID | Проверить HWID в системе, запросить новый файл |
| HWID меняется | Виртуальная машина | iRidium server не работает на VM |
| License corrupted | Повреждение файла | Скачать новый файл из облака |
| Не активируется триал | Уже использован / неверный аккаунт | Проверить аккаунт, создать новый |
